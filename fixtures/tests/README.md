## Overview

Use nvm to switch node versions.

### Node <15
- Node<15
- Node<15 + jsdom
- Node<15 + jest + jest-environment-node
- Node<15 + jest + jest-environment-node + jsdom
- Node<15 + jest + jest-environment-jsdom + jsdom

### Node >= 15
- Node>=15
- Node>=15 + jsdom
- Node>=15 + jest + jest-environment-node
- Node>=15 + jest + jest-environment-node + jsdom
- Node>=15 + jest + jest-environment-jsdom + jsdom
