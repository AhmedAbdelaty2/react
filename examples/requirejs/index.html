Example with JSX and RequireJS

To install React, follow the instructions on GitHub.

If you can see this, React is not running. You probably didn't run: 

          npm install -g react-tools
          jsx -x jsx . build/

Example Details

  • This is written with JSX and precompiled to vanilla JS by doing: 
              npm install -g react-tools
          jsx -x jsx . build/
  • AMD's define(…) and require(…) calls should include the React build as `React`, since jsx translations use `React`, e.g. <div>…</div> ↦ React.DOM.div(…). See app.jsx and its precompiled image, component.js.

Learn more at facebook.github.io/react.
