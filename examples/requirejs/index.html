Example with JSX and RequireJS

To install React, follow the instructions on GitHub.

If you can see this, React is not running. You probably didn't run: 

          npm install -g react-tools
          jsx -x jsx . build/

Example Details

  • This is written with JSX and precompiled to vanilla JS by doing: 
              npm install -g react-tools
          jsx -x jsx . build/
  • The React build's alias under define(…) and require(…) must be provided to the `@jsx` pragma. Supposing an alias for the React build of `x`: 
    /**
 * @jsx x.DOM
 */
define(['react'], function (x) {
  return reactjs.createClass({
    render : function () {
      return <div>Simple RequireJS Component</div>;
    }
  });
});
    See app.jsx and its precompiled image, component.js.
