Basic React Example with Webpack

To install React, follow the instructions on GitHub.

If you can see this, React is not working right. If you checked out the source from GitHub make sure to run grunt.

Example Details

This is basic example how to use React with webpack. Navigate to the example directory 

cd /path/to/react/examples/example-name
and run 
npm start

for development server run 
npm run dev-server

(don't forget to install first) 
npm install

Learn more about React at facebook.github.io/react.
